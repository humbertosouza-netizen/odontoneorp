# Odontoneo – Site Clínica Odontológica São José do Rio Preto

Site em **Next.js + React** focado em SEO local e conversão, com formulário de agendamento que envia mensagem automática para o WhatsApp.

## Como rodar

```bash
# Instalar dependências (se ainda não instalou)
npm install

# Desenvolvimento (abre em http://localhost:3000)
npm run dev

# Build para produção
npm run build

# Rodar em produção
npm start
```

## Estrutura da página

- **H1** – Tratamentos odontológicos em São José do Rio Preto – SP (SEO local)
- **Bloco de contato** – Endereço, WhatsApp e botão “Agendar avaliação” (acima da dobra)
- **Como funciona** – 4 passos da consulta (linguagem simples)
- **Tratamentos** – Implante, protocolo, próteses, canal, gengiva, reabilitação, odontopediatria
- **Bloco SEO local** – Dentista em São José do Rio Preto – SP
- **Formulário de agendamento** – Nome, idade, telefone, problema → mensagem pronta para WhatsApp
- **CTA final** – Contato e endereço
